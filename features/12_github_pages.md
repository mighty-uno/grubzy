# Feature Context: Showcase Landing Page & GitHub Pages Host

## Overview
Deploys a premium product showcase landing page to the repository root `index.html` to serve as the landing page for Zepkit on GitHub Pages.

---

## 1. Landing Page UI Specs
*   **Theme**: Dark Mode with Glassmorphism, neon highlights (`#E03E52` primary, `#3EB489` success green).
*   **Hero Callout**: Introduces Zepkit as the *Dopamine Delivery Simulator* (satisfying delivery cravings without physical calories or spending).
*   **Buy Me a Coffee CTA**: A gold-branded button link allowing visitors to support developer work.
*   **Screenshot Frames**:
    *   Responsive phone viewport mockups.
    *   Includes dynamic HTML/CSS previews of the app screens (Browse, Tracking countdown, Savings Analytics) if no physical screenshot image exists.

---

## 2. GitHub Pages Configuration
To serve the landing page:
1.  Navigate to your GitHub repository: `https://github.com/mighty-uno/zepkit`.
2.  Go to **Settings** -> **Pages**.
3.  Under **Build and deployment** -> **Source**: Select `Deploy from a branch`.
4.  Under **Branch**: Select `main` and set directory to `/ (root)`.
5.  Click **Save**.
